# Source Comparison Workflow

You are comparing multiple sources on the same topic. Analyze:

1. **Agreement Points**: Where do sources agree?
2. **Discrepancies**: Where do sources disagree?
3. **Complementary Findings**: Unique insights from each source
4. **Quality Assessment**: Relative quality and credibility
5. **Synthesis**: Combined understanding

## Comparison Report Format

### Source Comparison: [Topic]

**Sources Compared**: [List of sources]

**Areas of Agreement**: [Points where most sources agree]

**Areas of Discrepancy**: [Points of disagreement with analysis]

**Unique Contributions**: [What each source adds uniquely]

**Synthesis**: [Combined understanding based on all sources]

**Recommendation**: [Which source(s) to trust and why]
